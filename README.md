 Hunnab.Q (React + MySQL)

Proyecto con:

- Frontend React (`src/`)
- API Node/Express (`server/`)
- Conexión MySQL con `mysql2/promise` (`server/db.mjs`)



Instalar dependencias

```bash
npm install
```

Ejecutar en desarrollo

Terminal 1 (API):

```bash
npm run api
```

Terminal 2 (frontend):

```bash
npm start
```

- Frontend: `http://localhost:3000`
- API: `http://localhost:4000`
